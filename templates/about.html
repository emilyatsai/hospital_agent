About Kidney Stone Predictor

What is Kidney Stone Prediction?

Kidney stones are hard deposits made of minerals and salts that form inside your kidneys. This web application uses artificial intelligence to analyze urine analysis parameters and predict the likelihood of kidney stone formation.

How It Works
  1. Data Input: Enter urine analysis parameters (specific gravity, pH, osmolarity, etc.)
  2. AI Analysis: IBM Watson AutoML processes the data using machine learning algorithms
  3. Risk Assessment: Receive a prediction with confidence level
  4. Medical Consultation: Use results as a guide for professional medical advice
Technology Stack
  • Frontend: Flask web framework with Bootstrap UI
  • AI/ML: IBM Watson AutoML service
  • Data Processing: Python with pandas and scikit-learn
  • Deployment: Cloud-based Watson Studio
